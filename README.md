# 什么是灵犀体系
&emsp;&emsp;
如果说[ark宇宙](https://github.com/javaboy863/ark-universe) 是强大的炮火库及各种武器装备，
那么灵犀体系则是如何称心如意驱动这些武器的驱动程序，以下列表中的问题是需要在业务研发中重点考虑的。
